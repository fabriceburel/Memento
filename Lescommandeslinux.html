Les commandes de bases

Les commandes linux Explication de la commande
man Ouvrir le manuel d'une commande
clear Effacer le contenu du terminal
ls Afficher le contenu du dossier courant
ls -a ou ls -A Afficher le contenu du dossier courant avec les fichiers cachés
ls -l Afficher le contenu du dossier courant version longue sans les fichiers cachés
ls -la Afficher le contenu du dossier courant version longue avec les fichiers cachés
pwd Afficher le chemin du dossier courant
cd "nom du dossier" Se déplacer vers un dossier
sudo -s se mettre en super utilisateur
mkdir "nom du fichier" permet de créer un fichier
mv"nom du fichier" "nouveau nom du fichier" Permet de renommer un fichier ou un dossier
mv"nom du fichier""nom du dossier"/"nom du fichier" Permet de deplacer un dossier ou un fichier
touch"nom du fichier" Permet de créer un fichier
nano"nom du fichier" Permet d ouvrir un fichier
cp"nom du fichier à copier" "nom du nouveau fichier" permet de copier un fichier vers un nouveau fichier
Firefox"nom du fichier.html" permet d'ouvrir un fichier directement avec l'application désiré

Commande sur les paquets

Pour Rappel : La gestion de pacquet n'est faisable qu'en mode super utilisateur

1 paquet = un ensemble de fonction qui bout à bout permet de créer un logiciel

Les commandes linux Explication de la commande
apt update Mettre à jour la liste des paquets
apt upgrade Mettre à jour les paquets (permet de faire une mise à jour de l'ensemble des logiciel)
apt search XML Trouver tous les paquets contenant par exemple le mot XML
apt search apache | grep css Trouver tous les paquets contenant par exemple le mot apache en précisant la recherche sur le mot css
apt install GIT Installer installer un paquet (par exemple : GIT )
dpkg -i --force-overwrite atom-amd64.deb Permet de Lancer l'installation d'un .deb nouvellement téléchargé.

Si la dernière commande ne fonctionne pas effectuer les commandes suivantes:

  • apt --fix-broken install
  • apt upgrade
  • apt update
