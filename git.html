Sauvegarde de fichier avec git et github

Nom de la commande Explication de la commande
git init Permet d'initialiser un projet git
git add nom du fichier Permet l'indexation fichier suivi
git commit -m "nom de la sauvegarde" Permet la sauvegarde local des fichiers placé dans l'indexation
git status Permet de regardé l'état des fichiers
git diff Indique ce qui a été modifié dans un fichier suivi
git commit Permet de sauvegardé les fichier indexé et modifié
git branch "nom de la nouvelle branche" Permet de créer une nouvelle branche
git branch -d "nom de la branche que l'on souhaite supprimé" Permet de supprimer une branche
git checkout "nom de la branche que l'on souhaite être redirigé" Creation d'un alias origin afin de tranférer les donnée sur le lien github
le lien se récupère directement sur le site après sa création:
git remote add origin ""adresse http du lien"
git push origin "nom de la branche que l'on souhaite envoyer sur github" Permet de pousser la branche sur un depot distant tel que github
git pull origin "nom de la branche que l'on souhaite récupérer de github" Récupère et fusionne automatiquement une branche distante situé dans un dépot distant tel que github dans la branche locale
git merge "nom de la branche que l'on souhaite fusionné" Permet de fusionné une branche secondaire à la branche master
git fetch origin "récupère les données du projet sauvegardé sur gitHUB Permet de récupérer la branche que l'on souhaite à partir d'un depot distant tel que github
git clone git: emplacement du nouveau dossier de sauvegarde Permet de sauvegarder l'ensemble de mon dossier courant dans un nouveau dossier nommer "mongrit"
