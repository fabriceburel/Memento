Les bases

un fichier ou une section de code php se défini ainsi:

<?php

    ?>

pour créer une variable il suffit de l'écrire une première fois dans une boucle une fonction par exemple 

<?php
      $variable1=26;
      $varible = 'je met ce que je veux dans une variable';
    ?>

Les boucles for et while

Boucle for :

<?php
        for($var1;condition;incrementation) {
          {
            action;
          }
        ?>

Boucle while :

<?php
        $var1;
        while (condition) {
          action;
          incrementation;
        }
    ?>

$var1=initialisation de la variable

condition=condition qui arretera la boucle

incrementation=action qui doit être réalisé à chaque boucle

action=action désiré lorsque la condition qui arretera la boucle n est pas vrai
